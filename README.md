# start_vue_3

最简单的vue3组件库，就针对自己能用到的功能和属性

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
